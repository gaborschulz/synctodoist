Project

Bases: TodoistBaseModel

Project model

Attributes:

Name Type Description
name str

The name of the project
color ColorEnum | None

The color of the project
parent_id str | int | None

The id of the parent project
child_order int | None

The order of the project. Defines the position of the project among all the projects with the same parent_id
collapsed bool

Whether the project's subprojects are collapsed
shared bool

Whether the project is shared
sync_id str | int | None

Identifier to find the match between different copies of shared projects. When you share a project, its copy has a different ID for your collaborators. To find a project in a different account that matches yours, you can use the sync_id attribute. For non-shared projects the attribute is set to None
is_archived bool

Whether the project is marked as archived
is_favorite bool

Whether the project is a favorite
inbox_project bool

Whether the project is Inbox
team_inbox bool

Whether the project is TeamInbox
view_style str | None

A string value (either list or board). This determines the way the project is displayed within the Todoist clients.
Note

This class inherits all the properties of TodoistBaseModel
